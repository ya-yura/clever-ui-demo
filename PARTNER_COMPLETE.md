## 📋 Что было создано

Полнофункциональный модуль для совместной работы двух сотрудников на складе с отслеживанием статистики и управлением сессиями.

### ✨ Основные возможности:

- 👥 **Выбор напарника** из списка сотрудников
- 🤝 **Сессии совместной работы** с таймером
- 📊 **Статистика работы** (документы, строки, товары)
- 🎯 **Показатель совместимости** (0-100%)
- ⏸️ **Пауза/возобновление** сессий
- 📈 **История работы** с каждым напарником

---

## 🚀 БЫСТРЫЙ СТАРТ

### 1️⃣ Откройте приложение
```
http://localhost:3000
```

### 2️⃣ Перейдите к модулю
- Нажмите зелёную карточку **"🤝 Работа с напарником"** на главной
- Или откройте напрямую: `/partner`

### 3️⃣ Выберите напарника
- Используйте поиск 🔍
- Или выберите из недавних
- Нажмите **"✓ Начать работу"**

### 4️⃣ Работайте вместе!
- Статистика обновляется автоматически
- Используйте паузу для перерывов
- Завершите сессию по окончании работы

## 🔧 API Использование

```typescript
import { partnerService } from '@/services/partnerService';

// Начать сессию
const session = await partnerService.startSession(
  'user-001',     // Текущий пользователь
  'user-002',     // Напарник
  'receiving',    // Тип работы
  'doc-123'       // ID документа
);

// Обновить статистику
await partnerService.updateSessionStats(session.id, {
  documentsCompleted: 5,
  linesProcessed: 120,
  itemsProcessed: 350
});

// Завершить
await partnerService.endSession(session.id);
```
