🎓 BEST PRACTICES:
════════════════════════════════════════════════════════════════════

1. README.md в корне:
   • Первое, что видят на GitHub
   • Главная точка входа
   • Обзор проекта

2. DOCS/ для всего остального:
   • Детальная документация
   • Отчеты о работе
   • Инструкции
   • Архив изменений

3. Конфиги в корне:
   • package.json
   • tsconfig.json
   • vite.config.ts
   • .gitignore
   • и т.д.

4. Код в src/:
   • Исходники
   • Компоненты
   • Сервисы
