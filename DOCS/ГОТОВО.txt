❓ Списки документов пустые - это нормально?
✅ ДА! В dev режиме без API-сервера это нормально.
   Главное - что нет ошибок и интерфейс работает.

❓ Вижу предупреждения в консоли - это проблема?
✅ НЕТ! Предупреждения типа "Failed to load from API, using mock data"
   это нормально в dev режиме.

❓ Цифры на плитках случайные?
✅ ДА! В dev режиме счетчики генерируются случайно.
   С реальным API будут настоящие цифры.

❓ Как подключить реальный API-сервер?
✅ 1. Запустите Cleverence Mobile SMARTS сервер
   2. Убедитесь, что он доступен на http://localhost:9000
   3. Приложение автоматически начнет использовать реальные данные

  ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
───────────────────────────────────────────────────────────────────

1. Жесткая перезагрузка: Ctrl+Shift+R

2. Очистка кеша:
   F12 → Application → Clear storage → Clear site data

3. Переустановка:
   rm -rf node_modules
   npm install
   npm run dev

4. Проверка консоли:
   F12 → Console
   Ищите красные ошибки (не предупреждения!)

5. Читайте документацию:
   КАК_ЗАПУСТИТЬ.md - подробные инструкции
