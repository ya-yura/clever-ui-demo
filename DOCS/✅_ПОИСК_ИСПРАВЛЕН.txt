🔧 ТЕХНИЧЕСКАЯ РЕАЛИЗАЦИЯ:
════════════════════════════════════════════════════════════════════

Было:
```tsx
placeholder="🔍 Поиск по номеру..."  // Линза #1
<span>🔍</span>                       // Линза #2
```

Стало:
```tsx
placeholder="Поиск по номеру..."     // Нет линзы
{!searchQuery && <span>🔍</span>}    // Линза только когда пусто
```

Ключевые изменения:
• Убрали 🔍 из placeholder
• Добавили условие {!filter.searchQuery && ...}
• Добавили pointer-events-none (чтобы не мешала кликам)
