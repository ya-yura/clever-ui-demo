   ----------------------------------------

   2. Откройте вкладку Network

   4. Обновите страницу (F5)

   ----------------------------------------
   Chrome/Edge:
   • Settings → Privacy and security → Clear browsing data
   • Выберите "Cached images and files"
   • Нажмите "Clear data"
