⚠️ КРИТИЧНО: ОЧИСТИТЕ КЕШ И НАСТРОЙКИ
========================================

## Проблема

Старая конфигурация сохранена в localStorage и использует абсолютный URL вместо прокси.

## Решение

### Вариант 1: Через консоль браузера (БЫСТРО)

1. Откройте консоль: `F12` → вкладка "Console"
2. Выполните команды:

```javascript
localStorage.clear();
location.reload();
```

Это очистит все настройки и перезагрузит страницу.

### Вариант 2: Через DevTools Storage (РЕКОМЕНДУЕТСЯ)

1. Откройте `F12` → вкладка "Application"
2. В левом меню → "Storage" → "Local storage"
3. Кликните на `http://localhost:3002`
4. Нажмите кнопку "Clear site data" (или "Очистить данные сайта")
5. Обновите страницу: `Ctrl+Shift+R`

### Вариант 3: Через Diagnostics (В ПРИЛОЖЕНИИ)

1. Откройте гамбургер-меню (☰)
2. Выберите "Диагностика"
3. Нажмите "Очистить все данные"
4. Приложение перезагрузится автоматически

---

**СДЕЛАЙТЕ ЛЮБОЙ ИЗ ВАРИАНТОВ ПРЯМО СЕЙЧАС!**

```

```

А НЕ:
